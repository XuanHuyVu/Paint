User Instructions

Welcome to the Drawing App!

  • Use the size input to change the brush size.
  • Select a color using the color picker.
  • Use the erase button to switch to eraser mode.
  • Use the fill button to fill areas with the selected color.
  • Draw rectangles and circles using the respective tools.
  • Zoom in and out with the zoom buttons.
  • Undo and redo actions using the respective buttons.
  • Save your drawing in various formats using the save button.
  • Clear the canvas with the clear button.
  • Upload an image using the upload input.
